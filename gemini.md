모든 답변은 한국어로 함
모든 행위에는 설명이 동반해야 함. 석명의 수준은 프로그래밍 및 개발 초심자 수준에 맞춰서 해야 함.
PR은 항상 먼저 진행 여부를 물어봐야 함
PR시 타이틀과 내용은 한국어로 진행해야 함
guide.md 와 CONVENTIONS.md 파일을 매 행동에 우선 참조하여 최우선 행동 강령으로 삼아야 함
PR시 양식에 맞춰 PR 세부 내용을 복사할 수 있도록 출력해야 함
현재 여기서 해야할 일은 용현(Y) 파트임, 모든 행동은 해당 내용을 바탕으로 함.4
모든 답변은 ~입니다의 어미로 통일함
리포지토리 링크는 https://github.com/albenyu12/todorak.git 임.
guide.md, gemini.md, geminilog.md는 리포지토리에 업로드/추적하지 않고, 현재 개발 환경에서 임의로 수정하지 않아야 함.
모든 cli 사용 기록은 geminilog.md에 간단히 한국어로 기록해야 함. 포함되어야 하는 내용은 다음과 같음.
    1. 날짜/시간
    2. 유저 요청 요약
    3. 변경한 파일
    4. 변경점 요약
    5. 리포지토리와 상호작용한 내용 (없으면 없다고 표기)
    6. 실행한 명령어 목록
    7. 기타 실행한 점
    8. 오류가 있다면 오류와 잠재 해결 방안 / 해결했다면 어떻게 해결했는지
